# JMY科技 A股监控系统操作逻辑

## 🔄 整体流程

```
扫描系统 → 评分推荐 → 企微推送 → 决策买入 → 监控系统 → 自动提醒 → 决策卖出
```

---

## 1️⃣ 扫描系统 (stock-screener)

- **位置**: `/home/jmy/stock-screener/`
- **运行**: 每天收盘后自动扫描全市场
- **功能**: 筛选符合条件的股票（RSI、涨幅、量能等）
- **存储**: MySQL 数据库
- **前端**: http://localhost:8080/report
- **输出**: 信号列表（代码、名称、发现价、RSI）
- **新增**: 扫描完成后自动评分+企微推送推荐

### 评分规则（4分制）

| 条件 | 分值 | 
|------|------|
| RSI 60-75 | 1分 |
| 量比 > 2 | 1分 |
| MACD金叉 | 1分 |
| 非涨停 | 1分 |

| 总分 | 操作 | 仓位 |
|------|------|------|
| 4分 | ✅ 可进 | 20%满仓 |
| 3分 | ⚠️ 谨慎进 | 10%半仓 |
| ≤2分 | ❌ 不进场 | — |

入场价建议：现价 × 0.98（回调2%挂单）

---

## 2️⃣ 决策环节 (用户)

- 看扫描结果 → 判断是否买入
- 告知系统: "买入XX，价格XX，股数XX"
- 系统更新监控配置

## 3️⃣ 监控系统 (price_monitor.py v14)

- **位置**: `/home/jmy/.hermes/profiles/eastmoney-bot/scripts/price_monitor.py`
- **运行**: Cronjob 每分钟执行（交易时间 9:00-15:00）
- **配置**:
  - 总资金: 72,780元（原32,566 + 充值40,000 = 72,566，-沃顿997 +贤丰424）
  - 单股仓位: 20% = 14,513元
  - 持仓上限: 5个交易日
  - 默认止盈: +15%
  - 默认止损: -8%

### 监控内容

| 功能 | 说明 |
|------|------|
| 🔴🟢 涨停跌停 | 状态变化多次触发（9.95%阈值） |
| 📈📉 逼近涨跌停 | 涨跌幅≥8%时提醒 |
| ⚠️ 逼近止损 | 距止损≤3%每日提醒 |
| 🎯 逼近止盈 | 距止盈≤5%每日提醒 |
| 📈 整数涨跌幅 | -10%到+10%每个整数提醒 |
| 💰 成本线提醒 | 0.5%间隔提醒 |
| 🎯 止盈止损 | 触发自动提醒（优先检查） |
| 📊 RSI预警 | >90极度超买 / >80超买 / <20超卖 |
| 🚨 大盘风控 | 指数跌超3%暂停买入 / 跌超2%控制仓位 |
| ⏰ 持仓天数 | 超5天自动提醒 |
| 📊 整点播报 | 含持仓分析和建议 |

### 输出
- 企业微信推送
