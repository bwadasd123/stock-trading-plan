# 📈 3万本金股票交易计划

基于A股智能筛选系统的实战交易记录与操作计划。

## 基本参数

| 参数 | 值 |
|------|-----|
| 本金 | 30,000元 |
| 单股仓位 | 6,000元 (20%) |
| 买入方式 | 分两笔，各3,000元 |
| 止损线 | -8% |
| 止盈目标 | +15% |
| 持仓时间 | 3-5天 |

## 交易规则

1. **第一笔买足3000元**，不要保守
2. **回调2%再加第二笔**，不要追高
3. **持仓5天不管盈亏都减仓**
4. **止损-8%必须执行**
5. **止盈+15%分批卖出**
