# LOF/ETF套利标的知识库

## 概述
A股场内基金套利的核心逻辑：场内价格 vs 基金净值的价差机会。
常见模式：场外按净值申购 → 转托管到场内 → 场内高价卖出（溢价套利）。

## 套利关键特点
- **QDII额度限制** → 容易产生高溢价
- **场外限购/暂停申购** → 溢价持续
- **T+2/T+3时间差** → 套利有风险

## 套利流程
1. **发现溢价**：场内实时价格 vs 基金净值（T日收盘后公布）
2. **场外申购**：T日15:00前按净值申购
3. **等待确认**：T+2到账（QDII类可能T+3）
4. **转托管/卖出**：到场内卖出获利

## 风险点
- **时间差风险**：2-3天后才能卖出，期间溢价可能消失甚至变折价
- **净值波动**：底层资产价格波动
- **申购限额**：QDII额度紧张时可能限制或暂停申购
- **手续费**：申购费、赎回费、转托管费吃掉利润

## 监控需求
用户可能需要LOF/ETF溢价监控面板，实时追踪：
- 场内价格 vs 实时估算净值
- 溢价率排名
- QDII额度状态
- 申购限额提示

## 用户背景
用户有白银LOF（161226）套利实战经验，熟悉场内溢价场外申购的套利模式。
