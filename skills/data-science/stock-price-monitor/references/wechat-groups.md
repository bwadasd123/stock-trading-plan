# 企业微信群配置

用户有两个企业微信群，用途不同：

| 群 | Webhook Key | 用途 |
|----|-------------|------|
| 监控群 | `1d5118c7-ef37-4950-9a08-36da78763c7a` | 自动监控推送（cronjob） |
| 分析群 | `5af18435-5133-4583-ae4b-b71d38ae056f` | 手动分析报告推送 |

## 使用规则

- **监控群**：由 `price_monitor.py` cronjob 自动推送，内容包括价格预警、止盈止损、RSI、整点播报等
- **分析群**：用户主动要求时手动推送，内容为综合分析报告、买入建议、持仓分析等

## 调用方式

监控群 key 配置在 `.env` 中，脚本自动读取。

分析群 key 为手动使用，当用户说"推到另一个群/分析群"时，直接用完整 URL 发送：

```python
webhook = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=5af18435-5133-4583-ae4b-b71d38ae056f"
```
