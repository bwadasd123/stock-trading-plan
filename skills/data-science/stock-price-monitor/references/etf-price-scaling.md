# 东财API价格解析 — fltt参数

## 问题
东财push2delay API返回的价格字段（f43, f44, f45, f60）在不同`fltt`参数下格式不同：

| fltt | 格式 | 示例(股票002167) | 示例(ETF159599) |
|------|------|-----------------|-----------------|
| 1(默认) | 整数 | f43=2196 → 21.96需/100 | f43=3715 → 3.715需/1000 |
| 2 | 小数 | f43=21.98 → 直接用 | f43=3.715 → 直接用 |

## ⚠️ 核心教训
**fltt=1时除数不统一**：ETF(/1000) vs 股票(/100)，写统一脚本必出bug。

**2026-06-25实际案例**：
- 芯片ETF(159599): API返回f43=3715, 用/100得37.15(错), 用/1000得3.715(对)
- 东方锆业(002167): API返回f43=2196, 用/100得21.96(对), 用/1000得2.196(错)

## ✅ 解决方案
**统一用`fltt=2`，直接返回正确小数，无需任何除法。**

```python
url = f"http://push2delay.eastmoney.com/api/qt/stock/get?secid={secid}&fltt=2&fields=f43,f44,f45,f60,f170"
# f43=3.715 (ETF) 或 f43=21.98 (股票) — 直接用！
price = data["f43"]
change_pct = data["f170"]  # 3.11 表示 +3.11%
```

