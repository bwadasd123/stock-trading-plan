## 获取盘口深度数据

push2delay接口不返回买卖五档数据，需要：
1. 使用push2接口（需代理）
2. 使用网页端API
3. 使用APP截图+OCR

## 盘口分析要点

从东方财富APP盘口页面可获取：
- **委比**：(委买-委卖)/(委买+委卖)×100%，负数说明卖压大
- **内外盘**：外盘=主动买入，内盘=主动卖出
- **主力净流入**：主力资金当日净流入/流出
- **阶段涨幅**：3日/5日/10日/20日涨幅

**判断逻辑**：
- 外盘>内盘 → 主动买入多，看多
- 委比负数 → 卖盘挂单多，有压力
- 主力净流出 → 大资金在撤退
- 外盘>内盘但主力流出 = 主力在出货，散户接盘
