# 东财API字段映射（2026-06-23验证）

## push2delay接口字段

### ✅ 正确字段

| 字段 | 含义 | 解析方式 |
|------|------|----------|
| f43 | 现价 | ÷100 |
| f44 | 最高 | ÷100 |
| f45 | 最低 | ÷100 |
| f46 | 开盘 | ÷100 |
| f60 | 昨收 | ÷100 |
| f170 | 涨跌幅 | ÷100 |
| f47 | 成交量 | 直接用(手) |
| f48 | 成交额 | 直接用(元) |
| f161 | 内盘 | 直接用(手) |
| **f49** | **外盘** | **直接用(手)** ← 关键！ |
| f85 | 流通股本 | 直接用(股) |

### ❌ 错误字段

| 字段 | 错误含义 | 实际返回 | 真实值 | 问题 |
|------|----------|----------|--------|------|
| f51 | 量比 | 1902 | 0.71 | 字段含义错误 |
| f162 | 外盘 | 2920 | 55695 | 字段含义错误 |
| f52 | 振幅 | 1556 | 3.76% | 字段含义错误 |
| f50 | 换手率 | 73 | 7.79% | 需要计算验证 |

### 需要手动计算的指标

```python
# 振幅
amplitude = (f44 - f45) / f60 * 100

# 换手率
turnover = f47 * 100 / f85

# 外盘（直接用f49）
outer = f49

# 内外盘比
outer_ratio = f49 / (f49 + f161) * 100

# 量比（东财APP定义：当前成交量/过去5日同时段平均）
# 需要从K线计算，API不直接返回
```

## 量比计算方法

东财APP显示的量比 = 当前累计成交量 / 过去5个交易日同时段平均成交量

```python
# 近似量比（stock-screener使用）
approx_vol_ratio = today_volume / yesterday_volume

# 精确量比（需要分时数据）
# 1. 获取过去5天的分时成交量
# 2. 计算同时段平均值
# 3. 用当前成交量除以平均值
```
