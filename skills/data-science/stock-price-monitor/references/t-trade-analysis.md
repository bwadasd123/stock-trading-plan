# T交易分析框架

## 输入
- 当日卖出且亏损的股票
- 持仓成本（含历史亏损分摊）
- 30日K线数据

## 输出格式（简洁，不展开）

```
📊 收盘 X.XX | 成本 X.XX | 浮亏 +/-X

🎯 明日做T
| 操作 | 价位 | 逻辑 |
| 补仓 | X.XX | MA20+整数关 |
| 卖出 | X.XX | MA10压力 |

两种情况：涨→卖点减仓锁利，跌→补仓点加仓反弹出
```

## 关键指标计算

### 均线
```python
ma5 = mean(closes[-5:])
ma10 = mean(closes[-10:])
ma20 = mean(closes[-20:])
```

### 布林带
```python
bb_mid = ma20
bb_std = std(closes[-20:])
bb_lower = bb_mid - 2 * bb_std
bb_upper = bb_mid + 2 * bb_std
```

### 做T区间
- 买入：max(MA20, 整数关口) ~ max(MA20, 整数关口) + 0.05
- 卖出：min(MA10压力, 整数关口) ~ min(MA10压力, 整数关口) + 0.05
- T价差至少 0.15 才有操作价值
- 半仓操作，留余地

## 注意事项
- 实际成本必须含亏损分摊
- 到了目标价监控自动推送，不用盯盘
- 不要追涨加仓，不要在3毛以内空间操作
