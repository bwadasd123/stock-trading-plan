# 盘后T+1做T分析

## 触发条件
用户说"明天做T"、"盘后分析"、或当日有亏损卖出操作时，收盘后自动执行。

## 分析步骤

### 1. 确定分析标的
当日卖出且亏损的股票（考虑实际成本 = 买入均价 + 累计亏损÷当前持仓）。

### 2. 获取数据
- 东财API实时价格 + K线(30日) → 均线、布林带
- MA5/MA10/MA20 支撑阻力
- 周高低点

### 3. 计算关键价位

**成本计算（必须含亏损分摊）：**
```
实际成本 = 加权买入均价 + 历史累计亏损 ÷ 当前持仓股数
```

**做T区间：**
- 买入：MA20支撑位 ~ 整数关口（取较高者）
- 卖出：MA10压力位 ~ 整数关口（取较低者）

### 4. 输出格式
```
📊 收盘价 | 成本 | 浮亏

🎯 明日做T
| 操作 | 价位 | 逻辑 |
| 补仓 | X.XX | MA20+整数关 |
| 卖出 | X.XX | MA10压力 |

两种情况：
📈 涨 → 卖点减仓锁利
📉 跌 → 补仓点加仓，反弹出
```

## 注意事项
- 实际成本必须含亏损分摊，不要只用买入均价
- T价差至少0.15以上才有操作价值
- 半仓操作，留余地
- 到了目标价监控自动推送，不用盯盘
