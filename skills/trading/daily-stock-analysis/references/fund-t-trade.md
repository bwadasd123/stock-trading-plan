# 基金做T降本

## 基金 vs 股票做T区别

| 维度 | 股票 | 基金 |
|------|------|------|
| 买入确认 | T日即时 | T日收盘净值，T+1确认 |
| 卖出确认 | T日即时 | T日收盘净值，T+2到账 |
| 做T逻辑 | 当天买→当天卖旧份额 | 当天3点前买→次日确认→T+2赎旧份额 |
| 赎回顺序 | 自己选 | **先进先出（FIFO）** |
| 单日极限跌幅 | -10%（跌停） | 一篮子股票，-6%基本是极限 |
| 归零风险 | 有（退市） | **无** |

## 核心原理

用低净值份额替换高净值份额，总份额不变，成本下降。

```
做T = 买入低净值 → 反弹后赎回最早（成本最高）那批
```

## 计算公式

```python
# 补仓
新成本 = (旧股数 × 旧成本 + 新股数 × 买入净值) / (旧股数 + 新股数)

# 做T（反弹后赎旧份额）
T收益 = 赎回份额 × (赎回净值 - 买入净值)
新成本 = (旧成本×旧股数 + 买入净值×买入份额 - 赎回净值×赎回份额) / 旧股数
```

## 为什么不"等企稳"

基金不会退市。一篮子几十只股票，-6%单日跌幅基本是极限。连续两天-6%概率极低。

历史弹性参考：
- 015690: 7/9 +5.97%（急跌后反弹）
- 015690: 7/14 +4.36%（又急跌又反弹）

**基金急跌就是买点。** 不需要等"企稳"。

## C类赎回费规则

- 持有 ≤ 7天：1.5%惩罚费
- 持有 > 7天：0%
- 做T时必须确认赎的那批已超过7天

## 操作流程

1. 尾盘确认净值够低 → 买入（3点前）
2. 等反弹（通常2-5个交易日）
3. 净值反弹到目标 → 赎回最早那批（FIFO自动选成本最高）
4. 总份额不变，成本下降

## 实际案例：015690

```
原持仓: 1848份 @ 8.1174（投入15000，累计亏-1905）
做T:    买入706份 @ 7.086（投入5000）
反弹:   净值回到7.35
赎回:   赎6/29那批706份（成本最高≈8.3）
效果:   新成本 ≈ 7.92，T赚约186元
```
