# 资金管理 — 充值/入金歧义处理

## 常见歧义场景

当用户说"充值余额4w"、"冲了4w"等类似表述时，有两种理解：

| 用户表述 | 可能含义A | 可能含义B |
|----------|----------|----------|
| "充值余额4w" | 总资金 **设为** 4万 | 总资金 **增加** 4万 |
| "冲了4w" | 总资金 = 4万 | 总资金 += 4万 |

## 2026-07-02 教训

用户说"充值余额4w" → 我理解为总资金=40000
用户纠正："我是说我直接冲了4w" → 实际是 32566 + 40000 = 72566

## 正确做法

**遇到此类表述时，用一句话确认**，不要直接操作：

```
确认：总资金是 32566 + 40000 = 72566 对吗？
```

不要猜测，不要假设。涉及资金数字的错误修正成本高（需要回滚 commit）。
