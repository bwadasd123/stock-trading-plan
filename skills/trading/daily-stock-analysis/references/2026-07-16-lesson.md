## 根因分析

1. **分析框架有"第0步"但没有强制执行** — 7步框架写了"第1步:新闻 第2步:大盘"，但大盘-2.48%时我跳过了风险判断直接分析RSI
2. **技术指标绑架判断** — RSI 26.3、布林下轨、回调-38%这些数字让我觉得"机会来了"，忘了大盘才是最大的风险
3. **用户追问时没有坚持** — 用户反复问"挂单合理吗"，我每次都回答"合理"，应该在第一次就说"大盘弱，今天不适合"

## 修正后的规则（已写入SKILL.md）

```
回答"能买吗"之前，必须先过三关：

🛑 第0关: 大盘任一指數 ≤ -2%？
   → YES: "大盘偏弱，不适合买入，等企稳"
   → 到此为止。不分析RSI/布林/支撑位。

🛑 第1关: 大盘任一指數 ≤ -3%？
   → YES: "🚨大盘暴跌，暂停一切买入"
   → 到此为止。

✅ 第2关: 大盘全部 > -2%？
   → 通过，进入7步分析框架。
```
