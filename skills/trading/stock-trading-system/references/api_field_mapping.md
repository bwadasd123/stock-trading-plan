
## 📱 东财APP量比计算规则

东财APP显示的量比 = **当前累计成交量 / 过去5个交易日同时段平均成交量**

**示例**：
- 当前13:50成交量：10.49万手
- 过去5天同时段平均：14.77万手
- 量比 = 10.49 / 14.77 = **0.71**

**简化方案**（stock-screener使用）：
```python
# 近似量比 = 当日成交量 / 前一日成交量
approx_vol_ratio = today_volume / yesterday_volume
```

## 盘口深度分析

### 委比计算
```
委比 = (委买 - 委卖) / (委买 + 委卖) × 100%
```
- 委比 > 0：买方力量强
- 委比 < 0：卖方力量强
- 委比 > +50%：强烈看多
- 委比 < -50%：强烈看空

### 内外盘分析
- 外盘 > 内盘：主动买入多，看多
- 外盘 < 内盘：主动卖出多，看空
- 外盘 > 内盘 但 委比 < 0：可能是主力出货（挂卖单，散户主动买）

### 主力净流入
- 净流入 > 0：大资金在买入
- 净流入 < 0：大资金在卖出
- 连续5日净流出：主力在撤退，风险高

## 验证脚本

```python
import urllib.request
import json

def get_stock_data(secid):
    url = f"http://push2delay.eastmoney.com/api/qt/stock/get?secid={secid}&fields=f43,f44,f45,f46,f47,f48,f49,f60,f85,f161,f170"
    req = urllib.request.Request(url)
    with urllib.request.urlopen(req, timeout=10) as resp:
        data = json.loads(resp.read())["data"]
    
    return {
        "price": data["f43"] / 100,
        "high": data["f44"] / 100,
        "low": data["f45"] / 100,
        "open": data["f46"] / 100,
        "yesterday": data["f60"] / 100,
        "change_pct": data["f170"] / 100,
        "volume": data["f47"],           # 成交量(手)
        "amount": data["f48"],           # 成交额(元)
        "outer": data["f49"],            # 外盘(手)
        "inner": data["f161"],           # 内盘(手)
        "circulating_shares": data["f85"],  # 流通股本(股)
        "amplitude": (data["f44"] - data["f45"]) / data["f60"],
        "turnover": data["f47"] * 100 / data["f85"],
    }
```
