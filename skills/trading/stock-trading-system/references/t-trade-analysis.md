# T操作分析（做T降成本）

## 核心公式

### 手续费计算
- 买卖各5元/次，一趟T = 10元手续费
- 覆盖手续费所需最小波动 = 10元 / 股数

| 股数 | 最小波动 | 每0.01元净利 |
|------|----------|-------------|
| 100  | 0.10元 (0.28%) | -9元 |
| 200  | 0.05元 (0.14%) | -8元 |
| 300  | 0.03元 (0.10%) | -7元 |

### 补仓加权成本
```python
new_cost = round((old_shares * old_cost + new_shares * buy_price) / (old_shares + new_shares), 3)
```

## 做T策略选择

### 策略A：先卖后买（弱势/震荡市）
**适用条件**：RSI偏弱、均线空头、个股跌幅中
**操作**：等反弹到MA5附近卖出 → 等回落低位买回
**风险**：卖飞踏空（卖出后继续涨）

### 策略B：先买后卖（强势/上升市）
**适用条件**：RSI适中、均线多头、盘中急跌
**操作**：盘中急跌买入 → 等反弹卖出
**风险**：越跌越买（接飞刀）

## 决策因子

### 必须检查
1. **实时价格** — 用fltt=2 API获取
2. **RSI(14)** — 判断强弱：>60偏强适合先卖后买，<40偏弱谨慎操作
3. **均线排列** — MA5/MA10/MA20，多头适合先买后卖，空头适合先卖后买
4. **今日振幅** — 振幅<2%不适合做T（空间不够覆盖手续费）
5. **大盘环境** — 三大指数方向

### 不宜做T的情况
- 跌幅已大（-3%以上）：追低卖可能割在地板
- 振幅已走完大半：剩余空间不够
- 封板/逼近涨停跌停：流动性问题

## 实战案例
