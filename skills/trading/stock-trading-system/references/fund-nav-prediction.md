# 基金净值预测

## 天天基金实时估值API

基金当日净值通常晚间才公布，可用天天基金实时估值提前判断。

### API
```
http://fundgz.1234567.com.cn/js/CODE.js
```

### 示例：015690 富国中小盘精选混合C
```bash
curl -s "http://fundgz.1234567.com.cn/js/015690.js"
```

返回格式：
```json
jsonpgz({
  "fundcode": "015690",
  "name": "富国中小盘精选混合C",
  "jzrq": "2026-07-14",
  "dwjz": "7.7590",
  "gsz": "7.5938",
  "gszzl": "-2.13",
  "gztime": "2026-07-15 15:00"
})
```

### 字段说明
| 字段 | 含义 | 单位 |
|------|------|------|
| dwjz | 昨日单位净值 | 元 |
| gsz | 今日估算净值 | 元 |
| gszzl | 估算涨跌幅 | % |
| jzrq | 净值日期 | YYYY-MM-DD |
| gztime | 估值时间 | 收盘后为当日15:00 |

### ⚠️ 注意
- 仅交易日返回有效估值
- 估值基于基金季报披露的持仓加权计算，滞后于实际
- 权益类基金精度较高，债券/货币型精度低
- 9:30开盘即有数据，盘中每30秒更新
