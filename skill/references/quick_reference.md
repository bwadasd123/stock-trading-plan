# 交易快速参考卡

## 📊 核心参数
```
本金: 30,000元
单股: 6,000元 (20%)
买入: 分两笔，各3,000元
止损: -8%
止盈: +15%
持仓: 3-5天
```

## 🎯 RSI持仓策略
```
RSI 70-75 → 持仓3-5天 → 目标+15%
RSI 75-80 → 持仓2-3天 → 目标+10%
RSI >80   → 持仓1-2天 → 目标+8%
```

## 📋 每日检查清单
```
□ 盘后运行扫描器
□ 分析扫描结果
□ 准备次日资金
□ 设置条件单
□ 检查持仓盈亏
□ 更新交易记录
```

## ⚠️ 买入纪律
```
✓ 第一笔买足3000元
✓ 回调2%再加第二笔
✗ 不要追高加仓
✗ 高开>3%放弃
```

## 🚨 卖出规则
```
盈利>15% → 卖出50%
亏损>8% → 全部清仓
持仓5天 → 减仓50%
RSI>80顶背离 → 果断卖出
```

## 📱 条件单设置
```
止盈: 价格≥成本×1.15 → 卖出50%
止损: 价格≤成本×0.92 → 全部清仓
```
