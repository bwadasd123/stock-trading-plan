# 3万本金A股短线交易系统

> 适用于A股市场，基于技术指标扫描的短线交易策略

## 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 本金 | 30,000元 | 固定不动 |
| 单股仓位 | 6,000元 (20%) | 最大投入 |
| 买入方式 | 分两笔，各3,000元 | 降低风险 |
| 止损线 | -8% (-480元) | 必须执行 |
| 止盈目标 | +15% (+900元) | 分批止盈 |
| 持仓时间 | 3-5天 | 超时减仓 |

## 完整交易流程

### 第一步：扫描选股（盘后17:00+）

使用A股智能筛选系统扫描：

```bash
# 访问Web看板
http://localhost:8080

# 或使用API
curl "http://localhost:8080/api/scan/start" -X POST -H "Content-Type: application/json" -d '{"rsi_min":70,"rsi_max":80,"volume_ratio":1.5}'
```

**筛选条件：**
- RSI(14): 70-80（强势但不超买）
- MACD: 金叉确认
- 量比: >1.5（有资金关注）
- 价格: 站上MA20
- 板块: 资金流入

### 第二步：分析评估

获取扫描结果后，综合判断：

```python
# 获取实时价格
curl -s "http://push2delay.eastmoney.com/api/qt/stock/get?secid=0.{code}&fields=f43,f170,f44,f45,f46,f60"

# 分析要点
1. RSI位置 → 决定持仓时间
2. 量能 → 判断资金强度
3. 板块 → 确认热点方向
4. 价格位置 → 评估追高风险
```

**RSI与持仓对照：**
| RSI区间 | 持仓时间 | 目标收益 | 止损线 |
|---------|----------|----------|--------|
| 70-75 | 3-5天 | +15% | -8% |
| 75-80 | 2-3天 | +10% | -6% |
| >80 | 1-2天 | +8% | -5% |

### 第三步：次日执行买入

```
09:25 看集合竞价（高开<3%才做）
09:30 开盘观察
09:35 买入第一笔 3,000元
10:00 如回调2%，再买 3,000元
```

**⚠️ 关键纪律：**
- 第一笔必须买足3000元，不要保守
- 不要追高加仓，回调2%再加
- 高开>3%放弃操作

### 第四步：持仓管理

**每日收盘前检查：**
- 盈利>15% → 卖出50%
- 亏损>8% → 全部卖出
- 持仓5天 → 不管盈亏都减仓50%

**设置条件单（券商APP）：**
- 价格≥止盈价 → 自动卖出
- 价格≤止损价 → 自动清仓

### 第五步：价格监控

部署自动监控脚本，推送到企业微信：

```python
# 监控脚本位置
~/.hermes/profiles/eastmoney-bot/scripts/price_monitor.py

# 推送规则
- 开盘播报 9:30
- 整点播报 每小时
- 涨跌提醒 超2%
- 止盈止损 触发推送
- 收盘总结 15:00
```

**企业微信Webhook配置：**
```python
WX_WEBHOOK = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的key"
```

## 风险控制

### 止损规则
- 单笔止损: -8% 必须执行
- 每周止损上限: -2,400元 (-8%)
- 每月止损上限: -5,000元 (-17%)
- 达到止损线后停止操作3天

### 仓位规则
- 单只股票最多6,000元 (20%)
- 同时持仓最多3只
- 保留60%现金应对机会

### 时间规则
- 持仓最长5天
- 超时必须减仓50%
- 不要恋战，短线就是短线

## 收益预期

按每月20次操作计算：
- 胜率60%: 月赚约7,000元 (+23%)
- 胜率50%: 月赚约3,500元 (+12%)
- 保守估计: 月赚3,000-5,000元 (+10-17%)

## 实战教训

### 教训1：第一笔买太少
❌ 错误：100股≈1,600元（太保守）
✅ 正确：第一笔必须买足3,000元

### 教训2：追高加仓
❌ 错误：看到大单信号立即追高
✅ 正确：回调2%再加第二笔

### 教训3：不设止损
❌ 错误：扛单等回本
✅ 正确：-8%必须止损，不要幻想

### 教训4：频繁操作
❌ 错误：每天都想交易
✅ 正确：耐心等待信号，没信号不操作
